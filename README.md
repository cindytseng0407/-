# 桃園中壢 AVM 房價評估系統

## 快速開始（第一次安裝）

### 步驟一：安裝 Python

1. 前往 https://www.python.org/downloads/
2. 下載最新版（Python 3.11 或以上）
3. 安裝時 **務必勾選** "Add Python to PATH"
4. 安裝完成後，開啟「命令提示字元」（按 Win+R，輸入 cmd）
5. 輸入以下指令確認安裝成功：
   ```
   python --version
   ```
   應顯示 `Python 3.11.x` 之類的字樣

---

### 步驟二：安裝必要套件

在命令提示字元中，切換到本程式的資料夾：
```
cd C:\你存放的路徑\avm_system
```

安裝套件：
```
pip install -r requirements.txt
```

等待安裝完成（約1～3分鐘）。

---

### 步驟三：啟動程式

```
streamlit run app.py
```

瀏覽器會自動開啟 http://localhost:8501

---

## 日常使用

每次使用只需要：
1. 開啟「命令提示字元」
2. 切換到程式資料夾：`cd C:\你的路徑\avm_system`
3. 執行：`streamlit run app.py`
4. 用完關掉瀏覽器，回到命令提示字元按 Ctrl+C 停止程式

---

## 上傳真實資料（選用）

### 實價登錄資料
1. 前往 https://plvr.land.moi.gov.tw/DownloadOpenData
2. 選擇：桃園市 → 不動產買賣 → 最近 1～2 年
3. 下載 CSV 檔案
4. 在系統左側邊欄「資料來源」上傳

### 公告地價資料
1. 把 TXT 檔副檔名改為 .csv
2. 在系統左側邊欄上傳

上傳後系統自動使用真實資料計算，取代內建預設值。

## 三法估價說明

| 方法 | 預設權重 | 說明 |
|------|---------|------|
| 市場比較法 | 50% | 以商圈均價為基準，加入調整因子 |
| 成本法 | 25% | 地價分攤 + 建物折舊重置成本 |
| 收益法 | 25% | 租金收益直接資本化 |

權重可在左側邊欄自由調整（須合計100%）。

## 常見問題

**Q: 啟動後瀏覽器沒有自動開啟？**
A: 手動開啟瀏覽器，網址輸入 http://localhost:8501

**Q: 出現 `ModuleNotFoundError`？**
A: 重新執行 `pip install -r requirements.txt`

**Q: 如何更新內建商圈均價？**
A: 用文字編輯器開啟 `app.py`，找到 `DEFAULT_MARKET_PRICES` 修改數值。
